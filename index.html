How it works

The geeViz Agent connects a conversational AI (Google Gemini) to the full power of Google Earth Engine through the geeViz Python package. You type a question — the agent looks up datasets, writes and executes Earth Engine code, generates maps and charts, and presents the results directly in the chat.

Access is controlled via Google Sign-In. Contact the administrator to request access.
